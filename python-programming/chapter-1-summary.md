1. 프로그램은 컴퓨터가 할 작업을 담은 문서다.
2. 파이썬은 쉽고 편리하며 다양한 분야의 프로그래밍에 활용된다.
3. IDLE을 이용해 파이썬 프로그램을 만들 수 있고, 실행할 수 있다.
4. 대화식 쉘을 이용하면 프로그램 파일을 만들지 않고도 한 행씩 입력하여 실행 결과를 확인할 수 있다.
5. ``print()`` 함수는 값을 출력하고, ``input()`` 함수는 사용자에게서 값을 입력받는다.
